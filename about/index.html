Sobre o repetidores

Explorando repetidores de radioamador em Portugal com uma interface moderna e intuitiva

O que é este projeto?

Este projeto é uma ferramenta para explorar repetidores de radioamador em Portugal. Os dados são públicos, recolhidos da ANACOM e apresentados de forma interativa, permitindo aos utilizadores filtrar e localizar repetidores com facilidade.

Ver Código no GitHub

Funcionalidades

 

  • Visualização de repetidores em tabela com filtros avançados
  • Mapa interativo com localização geográfica
  • Interface responsiva otimizada para desktop e mobile
  • Filtros por indicativo, proprietário, banda e modulação

Sobre ser Radioamador

O radioamador, também conhecido como radioamadorismo, é uma atividade técnica e recreativa que utiliza ondas de rádio para comunicação. Os repetidores são estações que recebem um sinal numa frequência e retransmitem em outra frequência, ampliando assim o alcance das comunicações.

🎯 Objetivo: Facilitar a comunicação e promover o conhecimento técnico através da experimentação com equipamentos de rádio.
